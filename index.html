Neural Network from Scratch

Machine Learning

Custom multi-layer neural network in pure Python — no ML frameworks

Final project for an introductory machine learning course, built with Matan Moskowitz. A fully from-scratch multi-layer neural network — no TensorFlow, no PyTorch — implementing forward propagation, backpropagation, and gradient computation by hand, validated on MNIST and a breast cancer classification dataset.

  • Flexible architecture: arbitrary layer depth/width via a configurable layer_sizes array
  • Multiple activation functions: sigmoid, tanh, ReLU, leaky ReLU, softmax
  • Xavier (Glorot) weight initialization and mini-batch Stochastic Gradient Descent
  • Preprocessing pipeline: normalization, log transforms, one-hot encoding
  • ~98% accuracy on MNIST digits, ~80% on breast cancer classification

CollabEditor

Full-Stack Web App

Real-Time Collaborative Text Editor with Conflict-Free Sync

A React + CodeMirror editor where multiple users edit the same document simultaneously, with every keystroke merged in real time via Yjs CRDTs over WebSocket — no locking, no manual refresh. Live cursors and presence sync separately from document content via the Yjs Awareness protocol. Deployed as a single Node service that serves both the client and the WebSocket sync endpoint.

  • Real-time collaborative editing powered by Yjs CRDTs — conflict-free merges under concurrent edits
  • Live presence: colored cursors, selections, and an active-user list via Yjs Awareness
  • Persistent WebSocket sync — small binary diffs pushed to every client, not polling or full reloads
