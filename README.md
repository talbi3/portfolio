# Tal Bitton — Portfolio

A single-page portfolio site built with plain HTML, CSS, and JavaScript — no build step required.

## Sections

- **Hero** — animated typewriter role text, particle network background, social links
- **About** — background summary and quick stats
- **Skills** — categorized tech stack (languages, backend, systems, networking, databases, tools)
- **Projects** — [KafkaBoost](https://github.com/talbi3/Kafkaboost), [TRUST ME](https://github.com/talbi3/Trust-Me), [MTA-Coin](https://github.com/talbi3/BlockChain-threads)
- **Experience** — IDF Unit 81, QueenB Program
- **Education** — B.Sc. Computer Science, The Academic College of Tel Aviv-Yaffo
- **Contact** — email, phone, LinkedIn, GitHub

## Run locally

```bash
python -m http.server 4321
```

Then open `http://localhost:4321`.

## Deploy

This is a static site — deploy via GitHub Pages (Settings → Pages → deploy from `main` branch, root),
Vercel, Netlify, or any static host.
